FlopOS 0.1 - a kernel written from scratch 

to build FlopOS run: make
to run FlopOS run: make run

required packages (for debian / ubuntu / mint): build-essentials qemu-system gcc-multilib binutils grub-pc-bin xorriso qemu-system-x86 make

known bugs: command 'shutdown' not working on QEMU (not tested on bare metal) - weird symbol when first booted (run command clear to fix)
